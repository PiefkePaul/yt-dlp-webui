yt-dlp Download-Server

Selbst hostbarer Download-Server fuer yt-dlp mit Weboberflaeche fuer Einzelobjekte und Playlists ueber viele unterstuetzte Seiten hinweg.

Beispiele: YouTube, SoundCloud, Vimeo, Bandcamp, TikTok, Twitch und viele weitere Quellen, die yt-dlp direkt unterstuetzt.

Vor dem Download kannst du den Link analysieren und im erweiterten Menue Cookies, Header, Tokens, Referer oder extractor args nur fuer die aktuelle Session mitgeben.

Betriebsmodus
Erweitertes Menue fuer diese Session
Wird nur fuer diese Session verwendet. Bei SoundCloud-Go+-Tracks ist der Token noetig, damit nicht nur die 30-Sekunden-Vorschau geladen wird.
Optionales Seiten- oder Video-Passwort fuer diese Session.
Hilft bei eingebetteten oder referer-gebundenen Quellen.
Raw Cookie:-Header fuer diese Session. Praktisch, wenn du nur schnell Browser-Session-Cookies uebergeben willst.
Optional exportierter Cookie-Text fuer diese Session. Wird temporaer als Cookie-Datei an yt-dlp uebergeben.
Ein Header pro Zeile im Format Name: Wert.
Ein --extractor-args-Eintrag pro Zeile. So kannst du sitespezifische yt-dlp-Optionen pro Session setzen.
