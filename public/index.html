yt-dlp Download-Server

Selbst hostbarer Download-Server fuer yt-dlp mit Weboberflaeche fuer Einzelvideos und Playlists.

SoundCloud-Token

Benötigt für private, altersgeschützte oder Go+-Tracks.
Zu finden in den Browser-DevTools:
Application → Local Storage → soundcloud.com → oauth_token

Token wird nur im Browser gespeichert (localStorage) — nie dauerhaft auf dem Server.

Betriebsmodus
⚠ Kein SoundCloud-Token gesetzt — private oder Go+-Tracks können nicht geladen werden.
